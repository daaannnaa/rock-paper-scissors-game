# Rock Paper Scissors (Python)

A simple game where the user plays Rock–Paper–Scissors against a computer.

## Features
User vs computer gameplay
Random computer choice
Win / lose / tie outcome detection
Input validation (only accepts rock, paper, or scissors)

## How it works

Takes user input
Randomly selects a move for the computer
Compares both choices using game rules
Displays the result

## What I learned
Conditional logic (if/elif/else)
Random selection using Python
Handling user input
Designing simple game logic
